## 1. Purpose

To ensure all bookings are checked and handled accurately based on their status, ensuring proper communication with agents and hotels.

## 2. Scope

This procedure applies to all reservations processed by the front desk or booking management team.

## 3. Responsibilities

- **Front Desk / Booking Team**: Verify booking status, handle HCN checks, and communicate with agents and hotels.
- **Hotel Operations**: Respond to push requests when HCN is not OK.

## 4. Procedure

### Step 1 — Check Booking

- Verify booking details in the system and determine the current booking status.

#### Case A: Booking is **Cancelled**

- Inform the Agent (AGT) of the cancellation.
- Close the booking.

#### Case B: Booking is **Confirmed**

- Check Hotel Confirmation Number (HCN).
- **HCN is OK?**
  - ✅ **YES** — Follow the normal flow → Inform Agent → Close the booking.
  - ❌ **NO** — Push the request to the Hotel.
    - **Hotel responds OK**: Confirm with Agent → Close the booking.
    - **Hotel responds NOT OK**: Process Reject + Compensation (→ SOP-008) → Cancel and close the booking.

## 5. Notes

- Ensure timely responses to agents and hotels to avoid delays.
- All communication and status changes must be properly documented in the system.
- Always verify the HCN before proceeding with any confirmation to the agent.
- If the hotel does not respond, escalate to the supervisor immediately.

## Quick Reference

| 상황 | 액션 |
|---|---|
| Cancelled | Inform AGT → Close |
| Confirmed + HCN OK | Inform AGT → Close |
| Confirmed + HCN NOT OK + Hotel OK | Confirm AGT → Close |
| Confirmed + HCN NOT OK + Hotel NOT OK | Reject + Compensation (SOP-008) |
