## 1. Purpose

To ensure all incoming phone calls from hotels and agents are handled professionally and consistently. The caller is identified first, the correct booking number is requested, booking information is verified, and the appropriate action is taken based on the caller type and concern raised.

## 2. Scope

This procedure applies to all incoming phone calls received by the front desk or booking management team from hotels or agents (AGT) regarding any active booking.

## 3. Responsibilities

- **Front Desk / Booking Team**: Answer the call, identify the caller, request the correct booking number, verify booking details, handle the concern, and escalate or inform the relevant party.
- **Related Team / Supervisor**: Receive escalations or follow-up tasks as directed by the booking team.

## 4. Procedure

### Step 1 — Answer the Call with Standard Greeting & Identify the Caller

Answer the call immediately using the **VNOP standard greeting**:

> "Good morning / afternoon / evening, this is **[VNOP member name]** speaking, how can I assist you?"

Use the correct **time-based greeting**:

| 시간대 | 인사 |
|---|---|
| Before 12:00 PM | **Good morning** |
| 12:00 PM – 6:00 PM | **Good afternoon** |
| After 6:00 PM | **Good evening** |

Ask the caller to identify themselves:
- Are you calling from a hotel or as an agent (AGT)?
- If identity is unclear, ask for the company name or hotel name to confirm.

### Step 2 — Request the Booking Number

#### Case A: Caller is a **Hotel**

- Request the **ELLIS Booking Number** from the hotel.
- Pull up the booking in the ELLIS system.
- Verify: Guest name, Check-in / check-out dates, Room type and rate, Booking status.

#### Case B: Caller is an **Agent (AGT)**

- Request the **Agent Booking Number** from the agent.
- Pull up the booking in the system.
- Verify: Guest name, Check-in / check-out dates, Hotel name and booking status.

### Step 3 — Understand the Concern & Take Action

#### If Caller is **Hotel**:

- Listen to the hotel's concern (예: booking inquiry, guest issue, amendment, cancellation, or special request).
- Address and support the hotel's concern based on the verified booking details.
  - **Inform Agent (AGT)**: relay the hotel's message or update to the relevant agent.
  - **Inform Related Team**: escalate internally if the concern requires action from another department.
- Log all call details, the concern raised, and the action taken in the booking system.

#### If Caller is **Agent (AGT)**:

- Listen to the agent's request or concern (예: reconfirmation, name change, date amendment, cancellation, special request).
  - **Contact Hotel**: if the request requires hotel verification or action.
  - **Inform Related Team**: escalate internally if another department needs to act.
- **Refer to the relevant SOP** for the specific request type and proceed accordingly:
  - Reconfirmation → [SOP-003](./SOP-003.md)
  - Name change → [SOP-002](./SOP-002.md)
  - Free cancellation → [SOP-004](./SOP-004.md)
  - Date amendment → [SOP-005](./SOP-005.md)
  - Special request → [SOP-006](./SOP-006.md)
- Log all call details, the request, and the action taken in the booking system.

## 5. Notes

- **Always identify the caller before requesting any booking number** — never assume the caller type.
- Hotel callers must provide the **ELLIS booking number**; Agent callers must provide the **Agent booking number**. Do not proceed without verifying the correct number.
- All call details must be logged in the booking system immediately after the call, including: caller identity, booking number, concern raised, action taken, and parties informed.
- If the concern raised is outside the scope of VNOP, escalate to the relevant team promptly and inform the caller of the next steps.
- Always maintain a professional and courteous tone throughout the call, regardless of the concern raised.
- If the caller cannot provide a valid booking number, politely ask them to call back or offer to look up the booking using the guest name and check-in date as a secondary method.

## Caller Handling Quick Reference

| Caller | Booking Number Required | Next Action | Inform / Escalate To |
|---|---|---|---|
| **Hotel** | ELLIS Booking Number | Verify on ELLIS → Support concern | Agent or Related Team |
| **Agent (AGT)** | Agent Booking Number | Verify booking info → Determine next step | Hotel or Related Team |
